# Basic Colour Flipper|Palindrome Checker|To-Do List|Random Quote Generator|Stop Watch

A self challenge to make these as fast as possible, as simple as possible...

## Features

  Basic string logic
  Palindrome On-click events
  Local Storage in TO-DO list
  Non-repeatative quotes
  Perfectly working non glitchy Stop Watch
